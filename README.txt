# analysis_pipeline

All youll need to edit is the config.yaml, specify a name for the run (all of the files will be named with this), and what outputs you want. 
ex. runname: testrun1

# Output options

umap_plot: will generate a umap and embeddings file with default settings.

scib_metrics: will generate a csv of the scib metrics. 

crossgen_info: will generate crossgen umaps, summary statistics, and embeddings. 

ex. run_rules: all
